## Description

<p><span style="font-size: 18px;">Given two integers&nbsp;<strong>l</strong> and <strong>r</strong>, find the <strong>XOR</strong> of elements of the range [l, r].</span></p>
<p><span style="font-size: 18px;"><strong>Example:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>l = 4, r = 8 
<strong>Output: </strong>8 
<strong>Explanation: </strong>The value of (4 ^ 5 ^ 6 ^ 7 ^ 8) = 8.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>l = 2, r = 4<strong>
Output: </strong>5<strong>
Explanation: </strong>The value of (2 ^ 3 ^ 4) = 5.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>l = 5, r = 9<strong>
Output: </strong>5<strong>
Explanation: </strong>The value of (5 ^ 6 ^ 7 ^ 8 ^ 9) = 5.</span></pre>
<p><span style="font-size: 18px;"><strong style="font-size: 18px;">Constraints:</strong><br></span><span style="font-size: 18px;">1 ≤ l ≤ r ≤ 10<sup>9</sup></span></p>
